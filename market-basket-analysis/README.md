# 🛒 Market Basket Analysis & Recommendation System

## 📌 Overview

This project analyzes customer purchase patterns using Apriori and FP-Growth algorithms to generate product recommendations.

---

## 📁 Project Structure

```
market-basket-analysis/
│
├── data/
├── src/
├── requirements.txt
└── README.md
```

---

## ⚙️ Setup Instructions

### 1. Install Dependencies

```bash
pip install -r requirements.txt
```

### 2. Setup Kaggle API

* Download kaggle.json
* Place in:
  C:\\Users<your-username>\\.kaggle\\

### 3. Download Dataset

```bash
kaggle datasets download -d heeraldedhia/groceries-dataset
unzip groceries-dataset.zip -d data/
```

---

## ▶️ Run Project

```bash
cd src
python mba.py
```
