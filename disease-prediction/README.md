# 🩺 Disease Prediction System

A simple machine learning project that predicts diseases based on symptoms.

## 🚀 Features
- Uses Decision Tree Algorithm
- Simple CLI interface
- Easy to extend with real datasets

## 📁 Files
- model.py → trains and saves model
- utils.py → handles prediction logic
- app.py → user interface

## ⚙️ Installation
```bash
pip install pandas scikit-learn
```

## ▶️ Run

```bash
python model.py   # Train model
python app.py     # Run app
```

## 📌 Example

Input:
Fever: 1
Cough: 1
Fatigue: 1

Output:
Predicted Disease: Flu
