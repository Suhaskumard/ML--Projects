# 🚀 RecoSense Pro

Advanced Recommendation System using Collaborative Filtering.

## Features
- User-based & Item-based CF
- Train/Test Split
- RMSE Evaluation
- Top-N Recommendations
- CLI Interface

## Run

```bash
pip install pandas numpy scikit-learn
python app.py
```
